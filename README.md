# -Projet-MOM-Pizzeria
How to run and setup it

# New console project
dotnet new console --framework net7.0

// Run project
dotnet run

// Add package
dotnet add package

# GIT

// clone
git clone https://github.com/PaladinSkyland/Projet-MOM-Pizzeria

// Push to git
git add .
git commit -m "Message"
git push
